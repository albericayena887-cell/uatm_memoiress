# UATM GASA — Gestion des Mémoires Soutenus

Application web MVC pour la gestion, consultation, validation et archivage des mémoires soutenus (Université au Bénin).

## Démarrage rapide

1. Importer `database/uatm_memoires.sql` dans MySQL
2. Ouvrir `http://localhost/uatm_memoires/`
3. Se connecter : `admin@uatm.edu` / `password`
